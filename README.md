# Hang-Man-Game
This app makes you enjoy one of the most famous games played in notebooks, and its name is Hang Man. It is a guessing game for two or more players. One player thinks of a word, phrase, or sentence and the other tries to guess it by suggesting letters within a certain number of guesses. By typing your name, you would create an account for yourself, which allows you to see your progress.
